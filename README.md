# fab-scripts

Colección de scripts para automatizar tareas simples relacionadas con pruebas y procesamiento de datos.

## Instalación

1. Clonar el repositorio:
   git clone https://github.com/TU_USUARIO/fab-scripts.git

2. Entrar a la carpeta del proyecto:
   cd fab-scripts

3. Ejecutar el script deseado con Python:
   python nombre_del_script.py

## Ejemplo de uso

python saludo.py

## Requisitos
- Python 3.10 o superior
